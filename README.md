# FC THA — Spelersapp (concept)

Een verbeterd, mobiel-first concept van een spelers-monitoringapp voor een voetbalacademie.
Eén self-contained `index.html` — geen build, geen server, geen backend.

Alle gegevens zijn **voorbeeld-/demodata** en worden lokaal in de browser bewaard (localStorage).
Er wordt niets verstuurd.

## Onderdelen
Overzicht · Wellness · sRPE · Trainingsdata · Wedstrijddata · Lengte & gewicht · Testdata ·
Ontwikkelplan (POP) · Evaluatie · AcaFit · Weersadvies · Videoanalyse · Vakantie · Exporteren

## Gebruiken
Open `index.html` in een browser, of bekijk de live versie via GitHub Pages.
Op de telefoon: "Aan beginscherm toevoegen" voor een app-gevoel.

---
Concept/demo. Niet gelieerd aan officiële clubsystemen; alle data is fictief.
